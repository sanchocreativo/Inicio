Sancho fotografía

Este proyecto constó de la realización de mi página personal de fotografía, en el cual cuento actualmente como uno de mis grandes hobbies. Se decidió utilizar mi sobrenombre y la profesión de fotografía. La página consta de 3 secciones diferenciadas dependiendo del tipo de fotografía.

Creación de la marca

Creé un isologotipo con un tratamiento especial, elegido el color negro y rojo que es un color pregnante. Fué necesario crear un isologotipo que identificara con mayor preponderancia el atributo fotográfico.

Desarrollo de la página web

La página web fue construida en HTML, CSS y de framework fue utilizado Bootstrap 4. La página web es totalmente responsive, funciona tanto en dispositivos móviles como computadoras de escritorio
